# Global Life Expectancy Analysis

## Overview

This project explores global life expectancy trends and examines the key factors that influence population health outcomes across countries. The analysis combines data cleaning, exploratory data analysis, visualisation, and basic predictive modelling.

## Objective

The aim of this project is to:

* Identify the main drivers of life expectancy
* Compare developed and developing countries
* Analyse relationships between health and socio-economic indicators
* Build a simple model to estimate life expectancy

## Dataset

The dataset contains country-level information on life expectancy along with health, demographic, and economic indicators such as GDP, schooling, mortality rates, and population.

## Tools & Technologies

* Python (Pandas, NumPy)
* Data Visualisation (Matplotlib, Seaborn)
* Machine Learning (Scikit-learn)
* Power BI (Dashboard development)

## Data Preparation

* Cleaned column names for consistency
* Handled missing values using appropriate imputation methods
* Checked and corrected data types
* Selected relevant variables for analysis

## Exploratory Data Analysis

The analysis focused on understanding patterns and relationships in the data:

* Distribution of life expectancy across countries
* Comparison between developed and developing nations
* Correlation between key variables
* Visual exploration of trends and patterns

## Key Insights

* Life expectancy is significantly higher in developed countries
* Education (schooling) is one of the strongest predictors of life expectancy
* Adult mortality has a strong negative relationship with life expectancy
* GDP positively impacts life expectancy, with diminishing returns at higher levels
* Health-related indicators contribute to overall outcomes

## Predictive Modelling

A linear regression model was built using selected features such as GDP, schooling, and adult mortality. The model demonstrates how socio-economic and health factors can be used to estimate life expectancy.

## Dashboard

An interactive Power BI dashboard was created to:

* Compare countries and regions
* Explore relationships between variables
* Analyse trends over time
* Filter insights by development status
